# Frontend Instructions

## Stack

- **Framework**: React 18 + TypeScript
- **Build**: Vite
- **Styling**: TailwindCSS (dark gaming theme)
- **State**: Zustand with persist middleware
- **i18n**: i18next (French default, English available)
- **Animations**: Motion.js (framer-motion)
- **Panorama**: Pannellum for 360° viewer

## Structure

```text
frontend/src/
├── components/
│   ├── ui/         # Reusable UI (Button, Card, Input, Progress)
│   ├── game/       # Game components (PanoramaViewer, Timer, GuessInput)
│   └── layout/     # App layout (Header, LanguageSwitcher)
├── pages/          # Route pages (Home, Game, Leaderboard, Results)
├── stores/         # Zustand stores (gameStore, authStore)
├── lib/            # Utilities (cn, i18n config)
├── types/          # TypeScript type definitions
└── public/locales/ # Translation files (en/, fr/)
```

## Development

```bash
npm run dev      # Start dev server (port 5173)
npm run build    # Production build
npm run preview  # Preview production build
```

## Path Aliases

`@/` resolves to `src/` directory.

## Styling Conventions

- Dark theme: `bg-dark-*` classes (dark-900, dark-800, dark-700)
- Neon accents: `text-neon-purple`, `text-neon-pink`, `from-neon-purple to-neon-pink`
- Gaming button variant: `variant="gaming"` on Button component
- Use `cn()` from `@/lib/utils` for conditional classes

## State Management

```typescript
// Access store
const { score, currentTier, addScore } = useGameStore();

// Store persists to localStorage automatically
```

## i18n Usage

```typescript
import { useTranslation } from 'react-i18next';
const { t } = useTranslation();

// In JSX
<h1>{t('game.title')}</h1>
```

## Component Patterns

- Functional components only
- Use Motion.js for animations (`motion.div`, `AnimatePresence`)
- Import UI components from `@/components/ui`

## Adding Translations

1. Add key to `public/locales/fr/translation.json`
2. Add same key to `public/locales/en/translation.json`
